# shuoshubao.github.io

* git clone https://github.com/shuoshubao/shuoshubao.github.io.git
* cd shuoshubao.github.io
* npm install
* npm run dll
* npm start // 开发 [koa-webpack-middleware]
* npm run dev // 开发 [webpack-dev-server]
* npm run build // 上线
* npm run qiniu // 将静态资源发布到cdn上
* npm run eslint
