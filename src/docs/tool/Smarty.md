# 注释

```smarty
{*comment*}
```

# 变量

```smarty
{$name='shuoshubao'}
{assign var="name" value="shuoshubao"}
```

# 遍历

```smarty
{foreach $arr as $val}
{foreach $arr as $key => $val}
{foreach from=$arr item=v}
{foreach from=$arr item=v key=k}
```

# 重复

```smarty
{for $i=1 to 5}
```
